# Social Table Tennis Event Manager

A live event manager for social table tennis with 2 tables, 2v2 matches, and a flexible 10-game schedule.

## Core features

- Player management with statuses: Active, Resting, Left, Arriving later
- Automatic 2-table, 2v2 schedule generation for 10 rounds
- Balance player matches, subs, partner variety, opponent variety, and repeated pairing avoidance
- Lock matches once results are entered
- Regenerate only future/unplayed rounds from the current match
- Undo support for recent actions
- Live current match dashboard for fast result entry
- Full schedule, history, and final results views
